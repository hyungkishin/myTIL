### 제어 컴포넌트 (push / 진실의 원천)
- 리엑트 상태 === 사용자의 눈 === 브라우저 상태 (동기화)
- 개발자가 항상 동기화를 위해 푸시한다
(Handle Event)

### 비제어 컴포넌트 (Pull / 트리거 포인트)
- 개입하지 않다가 값이 필요할때 당겨온다
(가변 컨테이너 => Ref)
